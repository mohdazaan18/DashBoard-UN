# Code Review Agent Skill

## Backend Review

- Is CSV parsed only once?
- Is data stored in memory?
- Is filtering inside service layer?
- Are TypeScript interfaces defined?
- Is there only one endpoint?
- Are query params validated?

---

## Frontend Review

- Are components reusable?
- Is chart data dynamic?
- Is theme consistent?
- Is there no hardcoded mock data?

---

## General Review

- Is code readable?
- Is naming meaningful?
- Is structure aligned with decisions.md?
